   - start_device_auth() function: Returns device_code, user_code, verification_url, expires_in
   - poll_device_auth(device_code) function: Polls Google until user completes auth, returns token JSON
   - get_ytmusic_client(oauth_token_json) function: Creates authenticated YTMusic instance

3. Create src/types/youtube.ts with TypeScript interfaces:
   - DeviceAuthResponse: { device_code, user_code, verification_url, expires_in, interval }
   - AuthStatus: 'disconnected' | 'pending' | 'connected'
   - OAuthToken: { access_token, refresh_token, expires_at, token_type }
   - YouTubeTrack: { videoId, title, artist, album? }

<task type="auto">
  <name>Task 2: Add OAuth API endpoints to FastAPI</name>
  <files>api/index.py</files>
  <action>
1. Import youtube_music functions in api/index.py

2. Add POST /api/youtube/auth/start endpoint:
   - Calls start_device_auth()
   - Returns DeviceAuthResponse JSON (device_code, user_code, verification_url, expires_in, interval)

3. Add POST /api/youtube/auth/poll endpoint:
   - Accepts { device_code: string } in request body
   - Calls poll_device_auth(device_code)
   - Returns { status: 'pending' | 'complete', token?: string } where token is the JSON string to store
   - Returns status 'pending' if user hasn't completed auth yet (Google returns authorization_pending error)
   - Returns status 'complete' with token JSON string when user completes auth

4. Add CORS headers for frontend requests (add CORSMiddleware if not present)

Note: Do NOT add endpoints that require authentication yet - that's for Plan 02-03.
  </action>

  </verify>

2. Create src/components/YouTubeAuthButton.tsx:
   - Props: { onAuthenticated: (token: string) => void }
   - State: authStatus ('disconnected' | 'pending' | 'connected'), deviceInfo (user_code, verification_url)

   UI states:
   - disconnected: Show "Connect YouTube Music" button
   - pending: Show user_code prominently, verification_url as link, "Waiting for authentication..." message
   - connected: Show green checkmark with "YouTube Music Connected"

   Flow:
   1. User clicks "Connect YouTube Music"
   2. Call startAuth(), set authStatus='pending', store device info
   3. Start polling every 5 seconds (use interval from response)
   4. When poll returns 'complete', set authStatus='connected', call onAuthenticated(token)
   5. Clean up interval on unmount or completion
