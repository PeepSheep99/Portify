<task type="auto">
  <name>Task 2: Configure vercel.json for Python deployment</name>
  <files>vercel.json</files>
  <action>
Update vercel.json with proper Python serverless configuration:

```json
{
  "$schema": "https://openapi.vercel.sh/vercel.json",
  "functions": {
    "api/**/*.py": {
      "excludeFiles": "{tests/**,__pycache__/**,*.pyc,conftest.py,**/test_*.py}"
    }
  }
}
```

This configuration:
- Excludes api/tests/ directory from bundle
- Excludes __pycache__ directories
- Excludes .pyc compiled files
- Excludes conftest.py (pytest config)
- Excludes any test_*.py files

This keeps the bundle small and prevents test infrastructure from being deployed.
  </action>
  <verify>
